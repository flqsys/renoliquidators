Manage states/regions for multiple countries in WooCommerce checkout and shipping options.

== Description ==

States Manager For WooCommerce allows store administrators to enable or disable specific states/regions for different countries in their WooCommerce store. This is particularly useful for stores that:

* Only ship to specific states within countries
* Need to limit billing/shipping address options
* Want to customize their checkout experience based on geographical regions

Features:

* Easy-to-use interface for managing states/regions
* Search functionality for quick access to specific states
* Bulk selection/deselection of states by country
* Real-time counter for selected states
* Fully integrated with WooCommerce settings

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-states-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure WooCommerce is installed and activated
4. Use the WooCommerce->Edit States/Regions screen to configure the plugin

== Frequently Asked Questions ==

= Does this plugin work with the latest version of WooCommerce? =

Yes, the plugin is tested and compatible with WooCommerce 8.0.0.

= Can I restore states after disabling them? =

Yes, all state selections are saved and can be modified at any time through the admin interface.
